# Finanční plán - Filip Krayem

Samostatná stránka (self-contained `index.html`) finančního plánu pro Filipa Krayema.
Obsahuje pouze tento jeden plán - žádná cizí klientská data, žádné CRM.

- `index.html` - kompletní plán, CSS/fonty/obrázky inline (otevři v prohlížeči, funguje offline).
- Snímek z aplikace `financni-plany` (2026-06). Interaktivní prvky (varianty pojištění)
  jsou ve statickém snímku zobrazené ve výchozí variantě.

## Zveřejnění (volitelné)
- Vercel: import repa → Deploy (servíruje `index.html` v rootu).
- GitHub Pages: Settings → Pages → branch `main` /(root). Pozor: u veřejného repa
  by byl plán veřejně dostupný.
